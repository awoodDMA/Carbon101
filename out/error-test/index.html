Error Dialog Test Page

This page allows you to test different types of errors and see how the error dialog handles them. Each error will show a dialog with a copy button that formats the error information as compressed text.

Error Dialog Features:

  • • Shows detailed error information in organized sections
  • • Includes stack traces and component stacks
  • • Copy button formats everything as compressed text
  • • Removes extra whitespace and newlines for easy pasting
  • • Includes timestamp, URL, and user agent information
  • • Accessible with keyboard navigation and ARIA labels
